# A股潜在上涨股票预测分析系统

基于EastMoney数据源的A股市场预测分析工具，通过技术指标分析识别下一个交易日可能上涨的股票。

## 功能特性

### 1. 智能预测算法
- **多指标融合**: 综合RSI、MACD、移动平均线、布林带等多个技术指标
- **动态评分**: 基于技术信号生成预测分数
- **风险控制**: 考虑成交量和波动率等因素

### 2. 数据源兼容性
- **EastMoney优先**: 优先使用EastMoney实时数据
- **智能降级**: 当EastMoney不可用时自动使用高质量模拟数据
- **重试机制**: 实现3次重试逻辑应对网络波动

### 3. 可视化分析
- **预测排名**: 直观显示Top股票预测分数
- **技术指标**: RSI分布、价格走势等图表
- **热力图分析**: 多维度指标综合展示

## 预测指标详解

### RSI指标 (相对强弱指数)
- **超卖区** (≤30): 股票可能被低估，有上涨潜力
- **中性区** (30-70): 市场情绪平稳
- **超买区** (>70): 股票可能被高估，回调风险增加

### 移动平均线
- **短期均线上穿长期均线**: 看涨信号
- **价格突破短期均线**: 短期上涨动能

### MACD指标
- **MACD线上穿信号线**: 金叉，看涨信号
- **柱状图由负转正**: 多头力量转强

### 布林带
- **价格触及下轨**: 超卖，可能反弹
- **价格位于下半部分**: 有上涨空间

## 预测分数解释

| 分数范围 | 评级 | 解释 |
|---------|------|------|
| > 3 | 强烈看涨 | 多重技术指标发出买入信号 |
| 2-3 | 看涨 | 部分指标显示上涨潜力 |
| 1-2 | 中性偏多 | 整体趋势偏多头 |
| 0-1 | 中性 | 多空力量均衡 |
| < 0 | 看跌 | 技术指标偏向空头 |
