# 批量分析自选股功能增强

## 功能概述

本次更新为A股量化交易系统添加了以下重要功能：

### 1. 批量分析自选股
- 可一次性分析自选股列表中的所有股票
- 支持多种技术指标策略
- 提供详细的收益和风险指标
- 自动排序分析结果

### 2. 自选股管理
- 支持创建多个自选股列表
- 可以添加/删除股票
- 灵活的列表管理功能

### 3. 新增技术指标策略

| 策略名称 | 描述 |
|---------|------|
| Bollinger Band | 基于布林带的波动率和均值回归策略 |
| Mean Reversion | 基于统计属性的均值回归策略 |
| Breakout | 支撑阻力突破策略 |
| RSI Reversion | 专门针对RSI指标的均值回归策略 |
| EMA Cross | 基于指数移动平均线交叉的趋势策略 |
| Volume Spike | 基于成交量异常放大的策略 |

## 新增CLI命令

| 编号 | 命令 | 功能 |
|------|------|------|
| 18 | batch_analyze_watchlist | 批量分析自选股 |
| 19 | manage_watchlist | 管理自选股列表 |

## 使用示例

1. 在CLI界面中输入 `18` 或 `batch_analyze_watchlist` 来批量分析自选股
2. 输入 `19` 或 `manage_watchlist` 来管理自选股列表
3. 系统将引导您完成相应的操作

## 技术改进

- 增强了 StrategyManager 以支持更多策略
- 添加了 WatchlistManager 来管理用户自选股
- 扩展了 CLI 界面以包含新功能
- 添加了详细的帮助文档

## 策略总数

现在系统总共支持 **33种** 不同的交易策略，包括：
- 传统策略 (ma_crossover, rsi, macd 等)
- Qlib基础策略
- 高级多因子策略 (26+ 种)
